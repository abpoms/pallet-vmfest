A task for adding vmfest images

add-vmfest-image

fn

[request & args]

Install an image for vmfest.

For vagrant .box files, you will meed to specify the os-family, os-version, and os-64-bit, as these are not available in the .box file. add-vmfest-image image-url Usage:

Switches Default Desc
-------- ------- ----
-o, --os-family Specify the os family
-v, --os-version Specify the os version
-l, --os-64-bit Specify the os is 64 bit
